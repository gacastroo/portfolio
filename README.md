# Portfolio - React Developer

Portfolio personal desarrollado con React donde presento mis proyectos, habilidades técnicas y experiencia.  
El objetivo de este proyecto es mostrar mi capacidad para estructurar aplicaciones frontend escalables, reutilizables y bien organizadas.

---

## Sobre el proyecto

Esta aplicación está construida siguiendo buenas prácticas de desarrollo frontend:

- Arquitectura modular basada en componentes
- Separación clara entre layout, UI y páginas
- Reutilización de componentes
- Organización escalable de archivos
- Manejo dinámico de datos para proyectos

El proyecto refleja mi enfoque en código limpio, mantenible y estructurado.

---

## Stack Tecnológico

- React
- JavaScript (ES6+)
- CSS
- Vite
- Arquitectura basada en componentes

## Instalación y ejecución

Clonar el repositorio:
git clone https://github.com/gacastroo/Portfolio.git

Instalar dependencias:

npm install

Ejecutar en entorno de desarrollo:

npm run dev
